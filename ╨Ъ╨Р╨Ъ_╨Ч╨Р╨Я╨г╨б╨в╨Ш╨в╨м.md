# Автосбор омелы — как запустить

Скрипт находит омелу по цвету и форме и собирает её **двойным кликом** по кустику.
Картинки-шаблоны не нужны.

> ⚠️ Автоматизация нарушает правила онлайн-игр и может привести к бану.
> Используешь на свой риск.

## Команды по порядку

**1. Установить зависимости (один раз):**
```
pip install -r requirements.txt
```

**2. Проверить масштаб экрана:**
```
python herb_bot.py --check
```
В строке «Размер экрана» два размера должны совпадать (например, `1920x1080` и
`1920x1080`). Если РАЗНЫЕ — поставь масштаб дисплея 100% (Параметры → Система →
Дисплей → Масштаб = 100%) и перезапусти. Иначе клики промахиваются.

**3. Развернуть игру на весь экран, встать где растёт омела. Сохранить область:**
```
python herb_bot.py --grab
```
Открой `screen_region.png` — там должна быть только зелёная карта (без верхней
панели и чата). Если нет — поправь `GAME_REGION` в `herb_bot.py` или напиши мне.

**4. Проверить распознавание без кликов:**
```
python herb_bot.py --debug
```
Открой `debug_omela.png` — красные кружки должны стоять на омеле.

**5. Запустить сбор:**
```
python herb_bot.py
```
5 секунд на переключение в игру. Бот двойным кликом собирает омелу.

**Стоп:** курсор в левый верхний угол экрана или **Ctrl+C**.

## Настройки (вверху `herb_bot.py`)

| Параметр | Что делает |
|---|---|
| `GAME_REGION` | область экрана с картой |
| `GATHER_MODE` | `"double"` (двойной клик), `"click"`, или `"button"` |
| `OMELA_HSV_LOW/HIGH`, `BLOB_*` | настройка распознавания омелы |
| `GATHER_WAIT` | сколько ждать, пока идёт сбор |
| `MAX_PER_CYCLE` | защита от всплеска ложных срабатываний |
| `MAX_RUNTIME_MIN` | авто-стоп по времени |

## Если что-то не так

- **Клики мимо** → сначала `--check` (масштаб 100%), затем проверь `GAME_REGION` через `--grab`.
- **Двойной клик не начинает сбор** → возможно, сбор одиночным кликом: поставь `GATHER_MODE = "click"`.
- **Ловит лишнее / слишком много** → сузь `OMELA_HSV_*` и `BLOB_*`.
- Лог всех действий — в `herb_bot.log`.
