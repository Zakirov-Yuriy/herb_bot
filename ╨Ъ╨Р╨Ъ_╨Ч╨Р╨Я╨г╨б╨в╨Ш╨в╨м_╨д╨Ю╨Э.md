# Фоновый автосбор омелы (omela_bg.py)

Скрипт открывает **своё** игровое окно и кликает внутри страницы. Поэтому окно
можно свернуть и заниматься другими делами — бот продолжит собирать омелу в фоне.

> ⚠️ Автоматизация нарушает правила онлайн-игр и может привести к бану.
> Используешь на свой риск.

## Установка (один раз)

**1. Зависимости:**
```
pip install -r requirements.txt
```

**2. Браузер для Playwright (обязательно, один раз):**
```
playwright install chromium
```

## Первый запуск — вход в игру

**3. Открыть окно и войти:**
```
python omela_bg.py --login
```
- Откроется отдельное окно браузера. Войди в игру (логин/пароль), дойди до
  локации, где растёт омела.
- Вернись в терминал и нажми **ENTER** — сессия сохранится в папке
  `browser_profile` (в следующий раз вход не понадобится).

## Настройка (нужно сделать один раз со мной)

**4. Сделать скриншот от бота:**
```
python omela_bg.py --debug
```
Создадутся файлы `page_full.png`, `page_region.png`, `debug_omela.png`.
**Пришли мне эти три файла** — я по ним точно задам область карты (`MAP_REGION`),
чтобы бот смотрел только на карту и не задевал кнопки. После этого пришлю
обновлённый скрипт.

## Рабочий режим

**5. Запуск сбора:**
```
python omela_bg.py
```
Бот начнёт собирать омелу двойным кликом. **Окно можно свернуть** — сбор
продолжится в фоне, можно параллельно работать за компьютером.

**Стоп:** нажми **Ctrl+C** в терминале.

## Настройки (вверху `omela_bg.py`)

| Параметр | Что делает |
|---|---|
| `URL` | адрес игры |
| `VIEWPORT` | размер игрового окна |
| `MAP_REGION` | область карты внутри окна (зададим по скриншоту) |
| `OMELA_HSV_*`, `BLOB_*` | распознавание омелы |
| `GATHER_WAIT` | ожидание сбора |
| `MAX_RUNTIME_MIN` | авто-стоп по времени |

## Если что-то не так

- **`playwright install` ругается** — проверь, что интернет есть; команду запускать
  ровно как написано.
- **Окно открылось, но игра не загрузилась** — открой адрес игры вручную в этом окне,
  войди, потом нажми ENTER (для `--login`).
- **Бот кликает не туда** — значит надо поправить `MAP_REGION`; пришли мне `page_full.png`.
- Лог действий — в `omela_bg.log`.

> Примечание: `browser_profile` — это твой вход в игру. Никому не передавай эту папку.
