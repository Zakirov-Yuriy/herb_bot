# Автосбор ресурсов + авто-бой (omela_bg.py)

Бот открывает **своё** игровое окно и кликает внутри страницы (окно можно свернуть).
Собирает ресурс **выбранной профессии**, прокручивает карту, ведёт бой при нападении
монстра, закрывает окна-ошибки и не тыкает чужие ресурсы.

**Профессии:**

- `herbalist` — **травник**: омела/травы (жёлто-зелёный куст, режим по умолчанию).
- `geolog` — **геолог**: драгоценные камни/руда (несколько ярких цветов).
- `fisher` — **рыбак**: рыба (пресет-заготовка, лучше уточнить пипеткой).

Ресурс распознаётся по цвету. Для каждой профессии есть встроенный пресет цветов,
но **надёжнее снять цвет «пипеткой»** прямо из игры (Этап 0 калибровки) — особенно
для камней, которые бывают разного цвета.

> ⚠️ Автоматизация нарушает правила онлайн-игр и может привести к бану. На свой риск.

## Установка (один раз)

```
pip install -r requirements.txt
playwright install chromium
```

## Порядок настройки

1. **Вход в игру:** `python omela_bg.py --login` — войди, дойди до локации, ENTER.
2. **Профессия:** `python omela_bg.py --prof geolog` — выбрать, что собирать
2. **Профессия:** `python omela_bg.py --prof herbalist` --prof herbalist травник 
Сменить профессию — только через --prof <имя> (geolog / herbalist / fisher).
   (`herbalist` / `geolog` / `fisher`). Можно пропустить и выбрать в мастере (Этап 0).
3. **Калибровка:** `python omela_bg.py --calib` — пройди мастер (см. ниже). На Этапе 0
   выбери профессию и по желанию сними цвет камней/рыбы пипеткой.
4. **Работа:** `python omela_bg.py` — сбор + бой. Стоп: **Ctrl+C**.

> Профессия и снятый цвет сохраняются в `fight_zones.json`, так что настраивать это
> нужно один раз. При смене профессии старый снятый цвет сбрасывается автоматически.

## Мастер калибровки — `python omela_bg.py --calib`

Проходишь **этапы по порядку**, любой можно пропустить (ENTER без клика). Всё
сохраняется в `fight_zones.json`; можно запускать мастер несколько раз — уже снятые
точки не теряются.

**Этап 0. Профессия и цвет ресурса.** Выбери профессию (номер или имя:
`herbalist` / `geolog` / `fisher`). Затем — **пипетка**: укажи, сколько образцов
цвета снять, и кликай по нужному ресурсу в игре (для камней разного цвета — по
одному клику на каждый цвет). Бот запомнит эти цвета и будет искать только их.
Если снять `0` образцов — останется встроенный пресет профессии. Пипетка точнее
пресета, поэтому для камней/рыбы её рекомендуется пройти.

> Совет по пипетке: кликай **один раз ровно по центру** ресурса, затем ENTER. Если
> ресурсов много и разных — снимай несколько образцов, диапазоны суммируются.

**Этап 1. Область карты.** Кликни левый-верхний, затем правый-нижний угол зелёной
карты. Бот будет искать ресурсы только внутри этого прямоугольника.

**Этап 2. Способ добычи.** Введи в терминале, сколько кликов запускают добычу —
**1 или 2** (в этой игре — 2). Важно: бот больше не тыкает один и тот же куст
повторно, пока идёт добыча, поэтому добыча не отменяется сама.

**Этап 3. Кнопка «закрыть».** Кликни по чужому ресурсу (который твоя профессия НЕ
добывает), чтобы вылезло окно «Ошибка… нет профессии», и кликни его кнопку
**«закрыть»**. Дальше бот жмёт
эту точку сам — но только когда там появляется красная кнопка (т.е. окно открыто),
поэтому по пустой карте он не кликает и добычу не сбивает.

**Этап 4. Зоны боя.** Нужно быть **в бою** (видно колесо «БЛОК»). Кликни: зону
блока, зону атаки, «выход» (после победы), «В охоту». Нет боя сейчас — пропусти
этот этап (ENTER) и пройди его позже, когда будет бой.

## Что бот делает сам

- **Сбор:** находит ресурс выбранной профессии по цвету (пресет или снятый пипеткой),
  запускает добычу (`GATHER_CLICKS` кликов), не тыкает то же место повторно
  (`RECLICK_COOLDOWN` сек).
- **Прокрутка карты:** сканирует несколько положений по вертикали (`MAP_SCROLL_*`).
- **Окна-ошибки:** жмёт откалиброванную кнопку «закрыть», только когда она красная.
- **Чужие ресурсы:** клик привёл к ошибке → место запоминается и не кликается
  (`SKIP_FAILED_*`), отдельно на каждое положение прокрутки.
- **Бой:** блок + атака по раундам; победа → «выход» → «В охоту».

## Настройки (вверху `omela_bg.py`)

| Параметр | Что делает |
|---|---|
| `PROFESSIONS` | пресеты профессий: диапазоны цвета и форма ресурса (можно править) |
| `DEFAULT_PROFESSION` | профессия по умолчанию, если не выбрана (`herbalist`) |
| `GATHER_CLICKS` | сколько кликов запускают добычу (обычно из калибровки) |
| `RECLICK_COOLDOWN` / `RECLICK_RADIUS` | защита от отмены: не тыкать то же место |
| `MAP_REGION` | область карты (обычно из калибровки) |
| `BLOB_*` | форма пятна ресурса (задаётся профессией; ловит лишнее — сузь) |
| `POPUP_CLOSE_TARGET`, `POPUP_RED_FRAC` | кнопка «закрыть» и порог «красноты» |
| `POPUP_USE_GEOMETRY` | запасной авто-поиск окна (ВЫКЛ — может ложно кликать) |
| `SKIP_FAILED_*` | чёрный список чужих ресурсов |
| `MAP_SCROLL_*` | прокрутка карты |
| `FIGHT_*` | авто-бой и зоны колеса |

## Если что-то не так

- **Бот сам отменяет добычу** — проверь `GATHER_CLICKS` (этап 2) и что
  `POPUP_USE_GEOMETRY = False`; калиброванная точка «закрыть» не даёт ложных кликов.
- **Не закрывает окно «закрыть»** — пройди этап 3 калибровки. Проверить порог:
  открой окно и запусти `python omela_bg.py --debug` — в логе будет «доля красного
  в точке закрыть»; если < `POPUP_RED_FRAC`, уменьши порог.
- **Тыкает мимо ресурсов** — пройди этап 1 (область карты).
- **Не видит нужный ресурс / ловит лишнее (траву, фон)** — сними цвет **пипеткой**
  (`--calib`, Этап 0): кликай прямо по ресурсу в игре. Для камней разного цвета сними
  несколько образцов. Проверить, что бот сейчас видит: `python omela_bg.py --debug`
  (в логе — профессия и число найденных ресурсов).
- **Собирает не тот ресурс** — проверь профессию: `python omela_bg.py --prof <имя>`
  или Этап 0 калибровки.
- **Бьёт мимо в бою** — пройди этап 4.
- Лог — `omela_bg.log`.
